CodePad Help

Welcome to CodePad Help.
CodePad is a simple source code viewer that supports syntax highlighting. it supports all C-like, Bash-like, and XML-like languages. it automatically selects the syntax depending on the file.

CodePad uses WebKit and google-code-prettify to do the syntax highlighting.

Here are some Help:

1.View Code
You have two methods to view your code on your android device: 1.In "Home Page", click menu button, then select "open" to open the "File Browser". In file browser you can simply click the code source file to view its code(or long click the item and select "View Code" in the context menu). Remember that file without source code can't be viewed, CodePad will show you a dialog to tell you that he does not want to open for it not making sense. 2.In "Home Page", tap on the bottom of the screen, you will find that a drawer named "Project Home" is opened. In "Project Home", do as what you may do in "File Browser". Tips:".." in "File Browser" and "Project Home" means "up one level".

2.Search Code
You can search keyword in CodePad to view code efficiently. Do it like this: In "Home Page", press menu button, then select "Search". You will see a search dialog in the bottom of the screen.Search as what you like and what you used to. Press Exit button will close search dialog.

3.Select Code
You may want to share your code snippet with your friend(who is also a code monkey :) ) though sms or other method. To achieve this, just press menu button in "Home Screen", then select "select". Tap on the code you want to share, and CodePad will copy them to your clipboard. Just share as you want :).
